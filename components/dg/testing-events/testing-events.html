Notes

  • Events triggered by behavior $self can also be registered within the $public directive. However, if both $public and $private directives exist, events registered to the $private directive will take precedence.
  • To effectively run $dispatcher.emit() from within the grandchild view we have to first broadcast a parent-ready event from the parent component when it loads. We can then listen to the parent-ready event inside the grandchild view and run $dispatcher.emit() from its callback.
